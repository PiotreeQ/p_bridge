## Overview
pScripts Bridge is a Core library required for pScripts Resources. It provides essential functions and configuration options to ensure that scripts work seamlessly on your server.
## Features
- Configure your framework, target system, sound system, and other essential settings in one place.
- Plug-and-Play: Once set up, all our resources will work effortlessly without additional configuration.
- Open-Source: Modify and extend the library to fit your server's needs.
